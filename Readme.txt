NetScrape Extension

NetScrape is a browser extension designed to enhance your online security by detecting and blocking phishing attempts and malicious websites in real-time.

Features

Real-time phishing detection
User-friendly interface
Customizable settings
Regular updates for enhanced security
Installation

Download the netscrape.zip file from the Releases page.
Unzip the file.
Open your browser's extension settings.
Enable "Developer mode."
Click on "Load unpacked" and select the unzipped folder.
Usage

Once installed, NetScrape will run in the background and automatically scan websites for phishing attempts.
If a phishing attempt is detected, NetScrape will display an alert and block the website from loading.
